\subsection{Detailed Description}
\subsubsection*{template$<$typename T = float$>$class Green\+Func$<$ T $>$}



 \hyperlink{_greens_function_8h}{Greens\+Function.\+h} This class creates a map to represent Green\textquotesingle{}s function G(v, v0), so that distance, log, and sqrt have to be computed only once. Since, G(v, v0) = G(v0, v), only half of the matrix is saved. It is laid out linearly as a 1\+D map.

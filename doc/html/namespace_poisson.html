Detailed Description

Poisson.h Library for solving the Poisson equation using integral solution.

  • Technical details can be found in the paper doi: 10.1109/TVCG.2014.2312012

Author: Harsh Bhatia. bhati.nosp@m.a4@l.nosp@m.lnl.g.nosp@m.ovfu.nosp@m.nctio.nosp@m.ns for solving 2D and 3D Poisson equation on a regular grid f is called the source function that creates a potential (RHS) phi is the potential function (LHS)

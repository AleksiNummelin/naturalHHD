Detailed Description

template<typename T = float>
class GreenFunc< T >

GreensFunction.h This class creates a map to represent Green's function G(v, v0), so that distance, log, and sqrt have to be computed only once. Since, G(v, v0) = G(v0, v), only half of the matrix is saved. It is laid out linearly as a 1D map.
